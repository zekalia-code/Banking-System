# Banking Management System

## Project Description

The Banking Management System is a Java Console Application developed using Java, JDBC, and MySQL. It allows users to create bank accounts, securely log in using a mobile number and PIN, deposit money, withdraw funds, transfer money, and view transaction history. The project demonstrates CRUD operations, database connectivity, and object-oriented programming concepts.

## Features

Create Account

User Login

Deposit Money

Withdraw Money

Transfer Money

Check Balance

Transaction History

MySQL Database Integration
